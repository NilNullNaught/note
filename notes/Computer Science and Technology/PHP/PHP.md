## 1.2	PHP 基本概念

**PHP 脚本**

- PHP 脚本以 `<?php` 开始，以 `?>` 结束，可以放在文档中的任何位置。
- PHP 脚本在服务器上执行，然后将纯 HTML 结果发送回浏览器。

**php 文件**

- PHP 文件通常包含 HTML 标签和一些 PHP 脚本代码。PHP 文件的默认文件扩展名是 `.php`。

---

<br>

## 1.3	PHP Hello World

**例**

```php+HTML
<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
```

- 在 Web 服务器的 web 目录中创建 .php 文件即可，服务器将自动解析 php 文件。
